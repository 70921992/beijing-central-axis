## 五、已知问题与解决方案

### 问题 1：shejitan（社稷坛）图库图片过大
- **问题描述**：shejitan 目录下 5 张图片共约 29 MB，单张最大 8.49 MB
- **影响**：首次打开社稷坛详情时画廊图片加载缓慢，尤其在移动网络下
- **解决方案**：
  1. **推荐**：使用图片压缩工具（如 TinyPNG、Squoosh）将图片压缩至 200-500 KB/张
  2. 或将图片统一转换为 WebP 格式（浏览器兼容性：Chrome 23+, Firefox 65+, Safari 14+）
  3. `npm install -g sharp-cli && sharp -i *.jpg -o ./ --format jpeg --quality 80`

### 问题 2：file:// 协议下 Noto Serif SC 字体不可用
- **问题描述**：Google Fonts CDN 在 file:// 下无法加载
- **影响**：字体回退到系统默认衬线字体（视觉略有差异）
- **解决方案**：
  1. 下载字体文件放置于 `assets/fonts/` 目录，使用 `@font-face` 本地加载
  2. 或接受回退字体（对项目功能无影响）

### 问题 3：画廊关闭后定时器残留（已修复）
- **问题描述**：关闭详情卡片时未能停止 autoplay timer
- **解决方案**：`closeDetail()` 中已调用 `stopAutoplay()` ✅

### 问题 4：中轴线南段道路遗存 ID 不一致（已修复）
- **问题描述**：data.js 中 `nankuyicun` 与 gallery-data.js 中 `nanluyicun` 不匹配
- **解决方案**：统一为 `nanluyicun` ✅

### 问题 5：预加载双重回调（已修复）
- **问题描述**：安全超时回调可能和正常回调重复触发
- **解决方案**：增加 `done` 标志位和 `clearTimeout` 防护 ✅
